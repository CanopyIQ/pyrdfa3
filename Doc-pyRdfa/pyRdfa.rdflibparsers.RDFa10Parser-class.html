parse(self, source, graph, pgraph=None, media_type='')

source code 
Parameters:
  • source (InputSource class instance) - one of the input sources that the RDFLib package defined
  • graph (RDFLib Graph) - target graph for the triples; output graph, in RDFa spec. parlance
  • pgraph (RDFLib Graph) - target for error and warning triples; processor graph, in RDFa spec. parlance. If set to None, these triples are ignored
  • media_type (string) - explicit setting of the preferred media type (a.k.a. content type) of the the RDFa source. None means the content type of the HTTP result is used, or a guess is made based on the suffix of a file
  • rdfOutput (Boolean) - whether Exceptions should be catched and added, as triples, to the processor graph, or whether they should be raised.
